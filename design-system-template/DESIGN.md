# One Group Developers — DESIGN.md Master Overlay

> **What this file is:** The single source of truth for One Group branding.
> This is a **master overlay** — it applies on top of any base theme (Samsung One UI or future themes).
> The brand accent, canvas, typography, and logo are non-negotiable constants that override theme-specific accents.

## How to Use This File

This design system is split into sections. **Do not read all sections for every task.** Use the index below to open only the section you need.
